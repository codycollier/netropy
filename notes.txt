----------------------------------
:: design
----------------------------------

fundamental data:
- record values
- chain of records
- beacon as source

Possible usage from the client perspective:

#1
from netropy import beacon
b = beacon()


#2
from netropy import beacon

beacon.now()
beacon.current()

beacon.previous()
beacon.next()
beacon.last()

beacon.start-chain()
beacon.start()

record = beacon.now()


#3
from netropy import beacon
from netropy import record
from netropy import chain

rec = beacon.now()

record.verify(rec)

chain.verify_previous(rec)
chain.verify_next(rec)
chain.verify_pair(rec1, rec2)
chain.verify_all(rec)  <--- ?
